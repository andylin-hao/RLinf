快速开始
==========

欢迎使用 RLinf。本指南带你从安装到第一次训练运行，并指引接下来的步骤。

- **安装** — 使用 Docker 镜像或自定义环境（详见 :doc:`installation`）。
- **具身智能训练** — 在 ManiSkill3 上使用 PPO 训练 OpenVLA 与 OpenVLA-OFT（详见 :doc:`vla`）。
- **智能体训练** — 使用 GRPO 在 boba 数据集上训练 DeepSeek-R1-Distill-Qwen-1.5B（详见 :doc:`llm`）。
- **分布式训练** — 运行多节点的具身智能 / 智能体训练（详见 :doc:`distribute`）。

随手查阅 :doc:`cheat_sheet`，获取最常用的命令、环境变量与配置项。
