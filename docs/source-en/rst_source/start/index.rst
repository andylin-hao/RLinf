Get Started
===========

Welcome to RLinf. This guide takes you from install to your first training run,
then points you to what comes next.

- **Install** — use a Docker image or a custom environment (see :doc:`installation`).
- **Embodied training** — train OpenVLA and OpenVLA-OFT on ManiSkill3 with PPO (see :doc:`vla`).
- **Agentic training** — train DeepSeek-R1-Distill-Qwen-1.5B on the boba dataset with GRPO (see :doc:`llm`).
- **Distributed training** — run multi-node embodied / agentic training (see :doc:`distribute`).

Keep the :doc:`cheat_sheet` handy for the most-used commands, environment
variables, and config keys.
